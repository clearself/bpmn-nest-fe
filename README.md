# 一套高质量的vue3项目框架

技术栈：Vue3 + VueRouter + Typescript + vue-i18n + Ant Design vue + axios + pinia开箱即用；

功能：

- 可自定义主题
- 国际化
- 三种常见布局切换
- 高度完善的BPMN工作流可视化自定义流程实现，支持camunda、flowable、activiti引擎，可直接进行二次开发。

# 预览地址

预览演示地址：https://nest.flypa.cn/

# 安装

```bash
## install dependencies
pnpm install

## set up
pnpm run dev
```

## 构建

```bash
## build
pnpm run build
```

注：node版本是v20.8.0
